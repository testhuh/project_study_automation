Playwright 기반 콘텐츠 수집 · YouTube 플레이리스트 생성 · 결과 시각화 자동화

반복적인 콘텐츠 수집과 정리 작업을 자동화하고,
그 과정과 결과를 한눈에 확인할 수 있는
리포트 페이지로 정리했습니다.

📋 프로젝트 개요
프로젝트명
Project Study Automation
사용 기술
Python Playwright HTML CSS JavaScript GitHub Pages .env
구현 방식
일반 API 연동이 아닌
브라우저 자동화(Playwright) 기반
목표
반복 작업 자동화 및
결과 리포트 제공
🔄 자동화 흐름 단계
①
링크 콘텐츠 수집
▼
Playwright로 대상 페이지에 접속해 필요한 콘텐츠 정보(제목, 링크, 메타데이터 등)를 자동으로 수집합니다.
②
데이터 정리 및 가공
▼
수집한 원본 데이터를 정제하고, 다음 단계에서 활용할 수 있는 형식(CSV/JSON 등)으로 가공합니다.
③
YouTube 재생목록 생성
▼
가공된 데이터를 기반으로 각 항목을 검색·매칭하여 YouTube 재생목록을 자동으로 생성합니다.
④
HTML 리포트 생성
▼
실행 결과(성공/실패, 통계 등)를 시각적으로 확인할 수 있는 HTML 리포트를 자동으로 생성합니다.
⑤
GitHub Pages 결과 배포
▼
생성된 리포트를 GitHub Pages에 자동 배포하여, 별도 설치 없이 웹에서 바로 결과를 공유합니다.
🏆 주요 성과
✓
자동화 파이프라인 설계 경험
▼
수집 → 가공 → 생성 → 배포까지 이어지는 전체 흐름을 설계하고 직접 구현했습니다.
✓
Playwright 브라우저 자동화 구현
▼
실제 브라우저 동작을 코드로 제어하여 콘텐츠를 수집하는 자동화 스크립트를 구현했습니다.
✓
.env를 활용한 중요 설정 분리
▼
API 키, 경로 등 민감한 설정값을 코드와 분리해 안전하게 관리했습니다.
✓
GUI 실행 환경 구성
▼
비개발자도 쉽게 실행할 수 있도록 간단한 GUI 실행 도구를 구성했습니다.
✓
HTML 리포트 기반 결과 확인
▼
실행 결과를 표와 진행률 등 시각적인 형태로 정리한 HTML 리포트로 확인할 수 있습니다.
✓
GitHub Pages 렌더링 미리보기 제공
▼
별도 설치 없이 웹 브라우저에서 바로 결과를 확인할 수 있는 미리보기 페이지를 제공합니다.
